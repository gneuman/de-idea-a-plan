De Idea a Plan

Un plugin/skill para Claude Code que convierte una idea difusa en un plan claro y constructible en 10 minutos. Conversas con un agente, y al terminar tienes un HTML visual, un markdown editável, e instrucciones por etapa para que otro agente construya sin inventar.

Claude Code Plugin Conversacional MIT License Español

Qué hace

  • Conversación guiada

    El agente te pregunta paso a paso: propósito, qué hace, qué no va en v1, tech stack, modelo de datos, etapas. Una decisión a la vez.

  • Plan visual + markdown

    Genera HTML lindo (dark mode, imprimible), markdown editable, y prompt.md por etapa. Todo en _plan/.

  • Integración con Linear (opcional)

    /to-linear sube las etapas como milestones + tareas en tu workspace. Sin Linear igual funciona.

  • Fork-friendly

    MIT license. Cada skill tiene un bloque [CUSTOMIZE] para cambiar idioma, marca, stack default. Úsalo, forkéalo, véndelo.

Cómo está construido

1

Skills conversacionales

Dos skills en `.claude/skills/`: /idea (conversación + generación) y /to-linear (lectura + sync). Cada una se customiza en su propio [CUSTOMIZE] block.

Qué se incluye

  • /idea skill (flow completo)
  • /to-linear skill (MCP)
  • Templates HTML/markdown

NO incluye

  • App web propia
  • Base de datos

Done when: /idea corre, hace preguntas sensatas, y genera un HTML visual que parece profesional.

2

Distro + Docs + Comunidad

Publicar en plugin marketplace, mantener docs actualizadas, recopilar feedback de usuarios. Hacer que la herramienta sea conocida y fácil de adoptar.

Qué se incluye

  • README + ejemplos
  • Plugin marketplace
  • Taller live demo

NO incluye

  • Foro o comunidad propia
  • Support comercial v1

Done when: 50 personas han hecho al menos un plan con /idea, y hay feedback positivo sobre usabilidad.

Supuestos

  • Los usuarios tienen acceso a Claude Code y saben correr skills.
  • Tienen una idea de app, pero no un plan claro (ese es el problema que resuelve).
  • Linear es opcional; sin él la herramienta es igual de útil.
